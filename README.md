# Network Signal & Performance Monitor

Simple Python project simulating telecom network monitoring system.

## Features
- Simulates LTE/5G signal quality
- Logs network performance data
- Generates basic network report
- Classifies signal quality (GOOD / MEDIUM / BAD)

## Tech Stack
- Python
- Random module
- File handling
- Basic data analysis

## How to run
1. Run monitor.py
2. Run analyzer.py

## Output example
RSSI: -65 | SNR: 28 | STATUS: GOOD
